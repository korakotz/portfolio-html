จุดประสงค์ในการทำงาน

ต้องการเพิ่มเติมประสบการณ์การทำงานสายงานเกี่ยวกับ Programmer และการทำงานเป็นทีม

ประวัติการทำงาน

เคยทำงานเป็น Freelance รับทำเว็บไซต์ด้วยระบบ Wordpress ยังไม่มีประสบการณ์การทำงานร่วมกับบริษัท

ทักษะที่มี

PHP (ปานกลาง)

    • เคยทำระบบ user interface ด้วย API ของ Mikrotik (เป็น OS สำหรับจัดการระบบ Network) มีการ Create Read Update Delete ของระบบฐานข้อมูล

Laravel (ปานกลาง)

    • เคยทำระบบ Blog ด้วย Laravel การทำงานมีการ Create Read Update Delete ของระบบฐานข้อมูลได้

WORDPRESS (ชำนาญ)

    • เคยเขียนธีม WORDPRESS ด้วย Bootstarp 3 และมี Theme option ให้ใช้งาน
    • ทำเว็บด้วย Page builder ของ Divi และ Virtual Comporser (wp bakery)
    • ทำเว็บไซต์ร้านค้าออนไลน์ด้วย Woocommerce
    • ทำเว็บไซต์หลายภาษาได้
    • ปรับปรุงประสิทธิภาพให้เร็วขึ้น

HTML (ชำนาญ) CSS (ชำนาญ) Javascript (ปานกลาง)

    • ชำนาญในการใช้งาน HTML
    • CSS Framework ที่ถนัดคือ Bootstarp การปรับแต่งงส่วนต่างๆสามารถทำได้
    • Javascript สามารถนำโค๊ตสำเร็จรูปมาใช้งานได้
